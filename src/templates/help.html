Each (non-landing) page consists of the following sections.

Navigation Bar

The navigation bar is fixed to the top of the page and controls access to various sub-pages. Iconography used through the portal is explained elsewhere.


Jumbotron

The jumbotron is where flash messages are displayed. Routine messages are displayed in a pastel blue shade and error messages in pastel red.


Citation Header

The citation header reminds users of the purpose of the ARTN ORP and references the founding document(s).


Page Specifics

The page specifics section is dynamically created by various RESTful routes.


Footer

The footer shows the telescopes that make up the network and the site ownership.

This page is the front-facing page for user access. Only 2 actions are available from this page: Login and Register.


This page allows users to request an account on the ORP. When the form is submitted, and the inputs are verified, the user is sent an email notifying them that the moderator has been asked for approval for the new account. Once approval is granted, the user receives another email with an embedded login link.

Note that passwords must be a minimum of 8 characters and include at least 1 uppercase, 1 lowercase and 1 numeric character. The pass phrase has no such restrictions, just some memorable data, and is required along with a valid email address when requesting a password reset.


This page allows users access to their account, providing the account is not disabled. If account access is blocked, please contact the ARTN Operator. The 'remember me' button uses a browser-based token to keep the user logged in during a session. Please remember to logout when you are finished with the ORP.


This page indicates that the account is disabled. Please contact the ARTN Operator.


This page allows the user to request a password reset link be sent by email to the registered account. The pass phrase must be known. The link is valid for 600 seconds, after which it expires and the user must request another one.


This page is the user home page and provides buttons to create or view your observation requests, edit your profile and access help and API pages.


This page allows the user to change some aspects of their account profile.


This page allows the user to send appropriate feedback (with uploadable screen shots) to the ARTN ORP developers.


This page allows the user to upload multiple requests via a CSV or TSV-formatted file. If the file is < 100 entries, the upload is performed synchronously. Larger files are uploaded asynchronously in a separate thread and the user should refresh the screen to see new records arrive.

Only requests that are identified as belonging to the current user will be uploaded from within the file. If, however, the current user is an admin, all valid records will be created. A template TSV v2 file can be downloaded. Comment characters in this file include '#%!<>+-\/'.


This page allows users to create observation requests using a combination of telescope, instrument, dither, cadence, priority and other parameters.


This page shows tabular information derived from the complete, given, database record.


This page allows users to view all their observation requests. Admin users can view the requests of all registered users.
